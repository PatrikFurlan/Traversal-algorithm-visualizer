# Graph app

After learning about graph theory in our mandatory discrete structures class I got the idea of making an app to visualise and export graph files to help with visualising the most common graph algorithms

### Placing graph nodes

To place a node, left-click anywhere on the canvas. You can also move nodes around by dragging them with you mouse.

### Drawing edges between nodes

To begin drawing an edge between nodes, right-click a node. The node will change color which signals the node being selected and waiting for another node to be selected. After right-clicking the other node an edge appears between them.
